本篇博文主要记录学习mermaid的语法
mermaid是一款基于javaScript的画图插件，可以配合我的博客，通过在markdown中使用语法实现页面渲染。
我的博客基于Butterfly主题，该主题集成了mermaid插件
为了方便我今后的学习，觉得很有意义学习mermaid帮助我的笔记流程更流畅
故有此

markdown中使用mermaid

mermaid 语法

在mermaid中，所有图形都是由以下几个元素组成：

  • nodes
  • shapes
  • edges
  • arrows or lines
通用模板：
方向：

流程图可选择的方向：

  • TB（top to bottom - 从上到下
  • TD (top to down) 和TB一样
  • BT （bottom to top) 自下而上
  • RL （right to left)
  • LR (left to right)
节点形状:
  • round edges id1(xxxx)
  • stadium-shaped node id1([xxxx])
  • A node in subroutine shape id1[[ xxxx ]]
  • A node in a cylindrical shape id1[( database )]
  • A node in the form of a circle id1((circle))
  • A node in an asymmetric shape `id1>This is the text in the box]
  • rhombus id1{rhombus}
  • hexagon node id1{{hexagon node}}
  • parallelogram [/parallelogram/]
  • …
  flowchart TD;
    id1(round edges);
	id2([stadium-shaped node]);
	id3[[subroutine]];
	id4[(cylindrical)];
  
  flowchart TD;
	id5((circle));
	id6>asymmetric];
	id7{rhombus};
	id8;
	id9[/parallelogram/];
  • stadium-shaped node
连接各个节点
  • arrow A-->B
  • openLink A---B
  • text on links A-- text --B
  • text on arrow A-- text -->B
  • Dotted link A-.-B
  • Dotted link with text A-.text.->B
  • Thick link A ==> B with text A == text ==> B
  graph LR;
	A-->B---c-- text --> d-.-e ==> f;
网络
  • a & b 符号，表示这两个节点同时…
  graph TB;
	A --> B & C -.- D;	
	a & b --> c & d;
组合起来
  graph TD;
	A[Start]
	B{is it?}
	C[OK]
	D[ERROR]
	X[rethink]
	
	A-->B;
	B--> |yes|C--> X;
	X --> B;
	B ---> |No| D;
